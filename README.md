# VAPT Portfolio

This repository contains my hands-on practice in Web Application VAPT, 
focusing on OWASP Top 10 vulnerabilities.

## Skills Demonstrated
- Manual web application testing
- Authentication & Authorization testing
- Business logic flaw identification
- OWASP Top 10 & CVSS mapping

## Tools Used
- Burp Suite
- OWASP ZAP
- Nmap
- PortSwigger Web Security Academy

## Disclaimer
All testing is performed on intentionally vulnerable labs.
No real client or production systems were tested.
